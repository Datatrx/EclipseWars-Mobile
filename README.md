## Kurulum
1. Unity 6 LTS + Android Build Support kur
2. Bu repoyu klonla
3. Window > Package Manager > FishNet import
4. File > Build Settings > Android > Build

## Cloud Build
1. cloud.unity.com > DevOps > Build Automation
2. GitHub repo bağla
3. Android seç > Build Now
4. 20dk sonra APK hazır
